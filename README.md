# AI-CoPilot

AI-assisted development workflow prompts and templates.

## Structure

- `prompts/` - Collection of workflow prompt templates for AI-assisted development
  - `do-learning.prompt.md` - Repository learning methodology
  - `do-design.prompt.md` - Design documentation workflow
  - `do-prd.prompt.md` - Product requirements workflow
  - `do-codereview.prompt.md` - Code review workflow
  - `do-tests.prompt.md` - Test generation workflow
  - `do-acceptance.prompt.md` - Acceptance testing workflow

## Usage

These prompts are designed to be used with GitHub Copilot and other AI coding assistants to standardize development workflows.
